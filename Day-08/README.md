## Key Concepts Learned

### Class

A class is a blueprint used to create objects.

Example:

```python
class Student:
    pass
```

### Object

An object is an instance of a class.

Example:

```python
s1 = Student()
```

### Instance Variables

Variables belonging to a specific object.

Example:

```python
self.name
self.roll
```

### Constructor

A special method automatically called when an object is created.

Example:

```python
def __init__(self):
    pass
```
