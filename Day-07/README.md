# Day 07 - File Handling, Binary Files and Exception Handling

## Topics Covered

### 1. Text File Operations

* Reading files using `read()`
* Reading line-by-line using `readline()`
* Reading all lines using `readlines()`
* Writing to files using `"w"` mode
* Appending data using `"a"` mode

### 2. Context Manager

* Using `with open()` for automatic file closing
* Safer and cleaner file handling

### 3. File Navigation and Copying

* Using `seek()` to move file pointers
* Iterating through file contents
* Copying data from one file to another

### 4. Binary Files

* Writing binary data using `"wb"`
* Reading binary data using `"rb"`

### 5. OS Module

* Getting current working directory
* Changing directories
* Checking file existence

### 6. Pickle Module

* Serializing Python objects
* Writing records to binary files
* Reading records from binary files
* Handling EOF while reading records

### 7. Exception Handling

* Using `try-except`
* Handling specific exceptions
* Using generic exception handlers
* Understanding common Python exceptions

### 8. Advanced Exception Handling

* `else` block
* `finally` block
* Raising custom exceptions using `raise`

### 9. Assertions

* Validating conditions using `assert`
* Understanding `AssertionError`
