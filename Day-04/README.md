## Key Learnings

* Recursion allows a function to call itself.
* Every recursive function requires a base condition.
* Nested functions can access variables from outer functions.
* The global and nonlocal keywords modify variable scope behavior.
* Lambda functions provide a concise way to create anonymous functions.
* Functional programming tools such as map(), filter(), and reduce() can simplify data processing tasks.
* Recursive techniques can be applied to mathematical problems such as factorials, GCD calculation, prime number checking, and digit-sum computation.
