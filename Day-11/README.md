# Day 11 – Introduction to GUI Programming with Tkinter

## Overview

Today marked my first step into GUI (Graphical User Interface) development using Python's Tkinter library. Until now, all programs in the workshop were console-based, but today I learned how to create windows, buttons, forms, and interactive applications that users can interact with visually.

The exercises covered the fundamental Tkinter widgets and layouts that serve as the building blocks of desktop applications.

---

## Topics Covered

* Introduction to Tkinter
* Creating GUI Windows
* Labels
* Buttons
* Entry Widgets
* Event Handling
* Radio Buttons
* Check Buttons
* List Boxes
* Combo Boxes
* Message Boxes
* Grid Layout
* Pack Layout
* Form Creation

## Key Takeaways

* Tkinter allows Python programs to have graphical user interfaces.
* Widgets are the building blocks of GUI applications.
* Events make applications interactive.
* Layout managers help organize widgets inside windows.
* Forms combine multiple widgets to collect user information.
* Message boxes provide a simple way to communicate with users.
